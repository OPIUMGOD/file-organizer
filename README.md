## What it does

Scans your Downloads folder and moves everything into clean category folders:

`Images` `Videos` `Audio` `Documents` `Archives` `Programs` `Code` `Design` `Fonts` `Others`

Detects files by magic bytes first, not just extensions — so renaming a file won't fool it.  
Folders get analyzed too. If 60%+ of files are one type, the whole folder moves. If it's mixed, files get sorted individually.

---

## Usage

1. Run `Lz.exe`
2. Hit **Start**
3. Done

**Recursive scan** — enable this if you want it to go through subfolders too.  
A `log.txt` gets saved to your Downloads after every run.
